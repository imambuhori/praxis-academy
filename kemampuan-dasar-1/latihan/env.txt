	Software Ubuntu desktop yang saya pakai menerapkan gnome sebagai desktop environmet default. Untuk shell Ubuntu menggunakan terminal.
	Sedangkan window manager yang digunakan adalah jenis stacking yaitu sebuah window aplikasi bebas diatur ukurannya dan bebas dipindahkan sesuai keinginan atau kebutuhan.
Pada Ubuntu 18.04 LTS, display manager default sistem Ubuntu yaitu menggunakan GDM (Gnome Display Manager). GDM sendiri merupakan display manager default distro Linux Debian yang merupakan base-system dari Ubuntu. Projek untuk menerapkan GDM sebagai display manager default sistem Ubuntu 17.10 mungkin tidak akan tercapai dan baru bisa diterapkan pada Ubuntu versi 18.04. Sementara untuk desktop Gnome sudah dapat diterapkan pada Ubuntu 17.10.
